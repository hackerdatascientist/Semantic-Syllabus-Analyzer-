📝 MCQs:

Q1: What is the primary function of a Blockchain in a decentralized network?
 - To store and manage data reliably
 - To enable secure transactions between parties
 - To provide a global identity verification system
 - None of the above
✔ Answer: To enable secure transactions between parties
----------------------------------------
Q2: What is the name of the 53-qubit quantum computer developed by Google?
 - Ryerson University Computer
 - Sycamore quantum computer
 - IBM Cloud Quantum
 - Amazon Sagemaker
✔ Answer: Sycamore quantum computer
----------------------------------------
Q3: What type of machine learning algorithm is useful for solving problems where input data can only take on specific categories of values?
 - Reinforcement Learning
 - Decision Trees
 - Clustering
 - Supervised Learning
✔ Answer: Decision Trees
----------------------------------------
Q4: What is the concept in Blockchain that allows transactions to be made without a central authority?
 - Smart Contract
 - Mining
 - Consensus Mechanism
 - Cryptographic Hashing
✔ Answer: Consensus Mechanism
----------------------------------------
Q5: What type of network architecture makes use of resources offered by cloud service providers through APIs?
 - Hybrid Cloud
 - Infrastructure as a Service (IaaS)
 - Platform as a Service (PaaS)
 - Function as a Service (FaaS)
✔ Answer: Infrastructure as a Service (IaaS)
----------------------------------------
Q6: What is the primary difference between a cloud database and a distributed database?
 - Centralization vs Decentralization
 - Single-site vs Multi-site deployment
 - Cloud-based vs On-premises
 - Managed vs Unmanaged database
✔ Answer: Single-site vs Multi-site deployment
----------------------------------------
Q7: What is the type of network traffic management that involves using external servers that cache content for faster load times?
 - Load Balancer
 - Firewall
 - Content Delivery Network (CDN)
 - Web Server
✔ Answer: Content Delivery Network (CDN)
